### Run Locally

To run **Demo** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/ShayNeeo/Demo.git
```

Windows:

```bash
git clone https://github.com/ShayNeeo/Demo.git
```
